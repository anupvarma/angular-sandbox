# Finding-Falcone
This project is created with **Angular 11**. 

# Demo


# Build Setup

    #install dependencies
    npm install
    
    #serve with hot reload at localhost:4200
    npm start
    
    #build for production with minification
    npm run build
    
    #run unit tests
    npm test
    
    #run all tests and generate coverage report
    npm run test:coverage
